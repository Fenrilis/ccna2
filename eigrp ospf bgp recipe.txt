OSPF routing RECIPE:
multi vendor dynamic routing protocol
link state advertisment
implement djikstra algorithm
advertised by AREA (Area 0 is the backbone)

STATES:
D - Down
I - Init
T - Two-way
E - Extract
E - Echange
L - Loading
F - Full


recipe:
conf t
router ospf # (process id)
router-id (#.#.#.#) (most of the time are loopback add)
network (directly connected IP add) (wildcard of IP add) area (##)

network 10.1.1.8 0.0.0.3 area 34
network 4.4.4.4 0.0.0.0 area 34

BGP recipe:
BGP =  Border Gateway Protocol
Mostly used in WAN
AD = 20
Used by ISP and large Corporation

recipe:
conf t
router bgp # (AS No.)
bgp log-neighbor-changes
	neighbor (host ip ni kapitbahay) remote-as (AS no.)
	* no. of neighbors depends
	network (NetIP papunta sa kapitbahay) mask (subnet)
	* no. of network neighbors depends
end

R1:
conf t
router bgp 1
bgp log-neighbor-changes
neighbor 208.8.8.4 remote-as 45
neighbor 207.7.7.2 remote-as 2
neighbor 209.9.9.3 remote-as 3
network 208.8.8.0 mask 255.255.255.0
network 207.7.7.0 mask 255.255.255.0
network 209.9.9.0 mask 255.255.255.0
network 10.1.1.0 mask 255.255.255.0
end
